This page has been designed to let users upload csv files to interact with Copernicus Sentinel HUB or Google Earth Engine to train ML predictor and/or query SOC

QuerySOC screen access: .

When user needs to query SOC for a point, must follow this steps:

  • Select the ML predictor to use
  • Fill in query values:
    • Id: Point id.
    • Initial date: Format DD/MM/YY
    • End date: Format DD/MM/YY
    • Max cloud: Max cloud cover value.
    • Longitude: In WGS84.
    • Latitude: In WGS84.
    Once user push query button, SOC value will be shown on the text square on the right.
