New query access: .

This screen let users run queries to download geotiff files from Copernicus Sentinel HUB.

Parameters details are:
  • Reference:Reference
  • Initial Date:Initial date, format YYYY-MM-DD
  • End date:End date, format YYYY-MM-DD
  • Maxcloud coverage: Max cloud coverage for sentinel tiff images
  • Polygon: Coordinates format is lat<space>long,lat<space>long,,,,,,,,,lat<space>long
Screen gives user options to check polygon size. Copernicus Sentinel HUB does not support files with heigth or width greater than 2500. In case query does not find results next screen will be shown:
In case query find results it will be shown the list of .SAFE files. Check , to view tiff files.
