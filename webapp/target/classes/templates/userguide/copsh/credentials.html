Sentinel Credentials access: .

When launching queries to Copernicus Sentinel HUB is needed an authentication token.

If this token is missing, any action related to Copernicus Sentinel HUB menu will redirect to this screen.

To get this token user must input the clientID and the secret provided by Copernicus. User must follow this steps to get its own credentials:

  • Sing in into Dataspace Copernicus:
  • In the user's dashboard click on User settings
  • On the right side of the screen user will find section OAuth Clients. In case it does not exist, create a new one and do not forget to saved clientID and secret in a safe place.
