My queries list access: .

This screen let users list queries to download geotiff files from Copernicus Sentinel HUB.

For each query offers options to:
  • Show query details
    • Show query details, reference, initial and end date, max cloud cover and polygon. Coordinates format is lat<space>long,lat<space>long,,,,,,,,,lat<space>long User cannot edit this fields.
  • Show list of .SAFE files gotten when query was last time executed
    • For each .SAFE file user has access to download one geotiff per sentinel reflectance band, and show downloaded geotiff bands. After giving the order for downloading, an asynchronous process will be launched, so that user must wait and list later geotiff files.
    There are 2 different view options, as a map where polygon is shown or as geotiff image.
  • Relaunch query
  • Delete query
