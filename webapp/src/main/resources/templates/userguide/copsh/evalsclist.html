Evalscript list access: .

This screen let users list custom script to download geotiff files from Copernicus Sentinel HUB. An evalscript (or "custom script") is a piece of Javascript code which defines how the satellite data shall be processed by Sentinel Hub and what values the service shall return. It does exist a public repository for Sentinel Hub Custom Script where there are multiple examples created by researchers, that can be executed on WALGREEN. Gives users options to get custom script details screen and delete undesired items.
