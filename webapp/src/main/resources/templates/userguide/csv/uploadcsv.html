This page has been designed to let users upload csv files to interact with Copernicus Sentinel HUB or Google Earth Engine to train ML predictor and/or query SOC

User csv files upload access: .

When pushing "Choose file" button, WALGREEN will ask user to select a csv file.

Csv file must contain fields as shown on image using ";" as column separator.
Field details and specification is:
  • Depth: Indicate the depth used for digging to get soil samples.
  • POINTID: Internal ID that will be used to identify point.
  • OC: SOC value measured.
  • TH_LAT: WGS84 latitude.
  • TH_LONG: WGS84 longitude.
  • SURVEY_DATE: Date when sample was taken.
  • Elev: Elevation.
  • LC0_Desc: Description 1.
  • LC1_Desc: Description 2.
  • LU1_Desc: Description 3.
